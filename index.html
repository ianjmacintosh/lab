Showing a Footer on Mobile

The year is 2024, I'm adding a footer to a wicked simple page so it shows up on the bottom of the screen on desktop and mobile, and it's a lot harder than I expected.

If you're just looking for a solution: use #page-content { min-height: 100dvh; } (View live example)

If you want to learn more, I'll show you some other approaches that didn't do what I had in mind.

These pages are best viewed on mobile so you can see for yourself how each change affects the page display, but I'll include some screenshots from iOS and Android anyway.

  • Step 1: CSS grid (rows: 1fr auto) and min-height: vh100
  • Step 2: Remove the margin, add padding, apply box-sizing everywhere
  • Step 3: padding-bottom: calc(10px + env(safe-area-inset-bottom)
  • Step 4: Update the meta viewport tag with viewport-fit=cover
  • Step 5: Use min-height: 100dvh -- problem solved

Don't forget to add page code!

My Commentary

Welp, I guess we just have to live with it and if anyone asks, I'll need to explain that browsers get the last word on how pages render.

But wait a second, there's yet another solution. And this one actually makes sense: set the page height using dvh units.

"Device Viewport Height" is the height of the viewport, not including the height of the browser chrome. (Source: CSS Values and Units Module Level 4)

It seems like browser authors have given us exactly what we want.
